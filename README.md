# 🌐 Promoting Temporal Diversity in Recommender Systems via Explanations  

This repository contains the code and experimental assets for the paper submitted to ACM TOIS (2025).

# ✨ Overview  
This project introduces a framework to enhance **long-term diversity** in recommender systems by leveraging **explainable AI techniques**.  
The method tracks how recommendations evolve across multiple interactions and uses **reasoning paths** on a knowledge graph to identify items that may reduce future diversity. Corrective actions are then applied to maintain high diversity without compromising accuracy.  

# 🧩 Key Features  
- **Explainable Graph-Based Recommenders** (PGPR, CAFE)  
- **Segregation Score** to measure an item’s impact on diversity  
- **Corrective Actions:** Re-Ranking (RR) and Modify Weights (MW)  
- **Multi-Iteration Evaluation** of recommendation dynamics  

# 📈 Results Summary  
- Up to **+18.3% diversity gain** with minimal accuracy loss  
- Sustains higher diversity across multiple iterations compared to baselines
